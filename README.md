
# Encuesta Ciudadana Nacional - Perú (Versión 3.0)

Incluye:

- Login con Google
- 1 voto por usuario
- Resultados en tiempo real
- Panel administrador privado (por email)
- Exportación a CSV (abrible en Excel)
- No se recopilan datos sensibles

============================
CONFIGURAR FIREBASE (PASO A PASO)

1. Ir a https://console.firebase.google.com
2. Crear proyecto nuevo
3. Ir a Authentication → Activar Google
4. Ir a Firestore Database → Crear base en modo producción
5. En Configuración del proyecto → General → Añadir app web
6. Copiar datos en firebase-config.js
7. Reemplazar TU_CORREO_ADMIN@gmail.com por tu correo

Luego subir a GitHub y activar GitHub Pages.

IMPORTANTE:
No compartas tu correo admin públicamente.
No modifiques votos manualmente.
Mantén transparencia metodológica.
